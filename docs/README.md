# docs/

This directory is intentionally empty in version control.

## Source book

This project is a paraphrased companion to **Charles Platt's *Make: Electronics*, 3rd Edition** (Make Community, 2021). The site assumes you own a copy of the book — none of the book's prose, photos, or illustrations are reproduced here.

If you want to read along while exploring the experiments, put your personal copy of the PDF (or any other format) here as `Make - Electronics (3e).pdf`. The file is gitignored so it stays local-only.
